### Дата: 2024

## Часть 1: Выбор тестируемой функции

Выбраны для тестирования все ключевые функции калькулятора:
- `add(a, b)` - сложение двух чисел
- `subtract(a, b)` - вычитание
- `multiply(a, b)` - умножение
- `divide(a, b)` - деление
- `power(a, b)` - возведение в степень

**Основная функция для детального тестирования:** `divide()` - функция деления, так как она имеет важную обработку ошибок (деление на ноль).

---

## Часть 2: Написанные тесты

### Файл: `test_calculator.py`

Создан тестовый класс `TestCalculator` с использованием модуля `unittest` (аналог JUnit для Python).

### Общее количество тестов: 26 тестов

#### Тесты функции `add()` (сложение) - 5 тестов:
1. `test_add_positive_numbers` - сложение положительных чисел
2. `test_add_negative_numbers` - сложение отрицательных чисел
3. `test_add_mixed_numbers` - сложение с разными знаками
4. `test_add_with_zero` - сложение с нулем
5. `test_add_float_numbers` - сложение дробных чисел

#### Тесты функции `subtract()` (вычитание) - 5 тестов:
1. `test_subtract_positive_numbers` - вычитание положительных чисел
2. `test_subtract_negative_numbers` - вычитание отрицательных чисел
3. `test_subtract_mixed_numbers` - вычитание с разными знаками
4. `test_subtract_with_zero` - вычитание с нулем
5. `test_subtract_float_numbers` - вычитание дробных чисел

#### Тесты функции `multiply()` (умножение) - 5 тестов:
1. `test_multiply_positive_numbers` - умножение положительных чисел
2. `test_multiply_negative_numbers` - умножение отрицательных чисел
3. `test_multiply_mixed_numbers` - умножение с разными знаками
4. `test_multiply_with_zero` - умножение на ноль
5. `test_multiply_with_one` - умножение на единицу

#### Тесты функции `divide()` (деление) - 6 тестов:
1. `test_divide_positive_numbers` - деление положительных чисел
2. `test_divide_negative_numbers` - деление отрицательных чисел
3. `test_divide_mixed_numbers` - деление с разными знаками
4. `test_divide_float_result` - деление с дробным результатом
5. `test_divide_by_one` - деление на единицу
6. **`test_divide_by_zero`** - **деление на ноль (проверка исключения ValueError)** ⭐

#### Тесты функции `power()` (возведение в степень) - 5 тестов:
1. `test_power_positive_exponent` - возведение в положительную степень
2. `test_power_zero_exponent` - возведение в степень 0 (результат = 1)
3. `test_power_one_exponent` - возведение в степень 1
4. `test_power_negative_exponent` - возведение в отрицательную степень
5. `test_power_fractional_base` - возведение дробного числа в степень

---

## Что проверяют тесты:

1. **Корректность базовых операций** - проверка правильности математических вычислений
2. **Работа с положительными числами** - стандартные случаи использования
3. **Работа с отрицательными числами** - проверка правильности знаков
4. **Работа с нулем** - граничные случаи
5. **Работа с дробными числами** - точность вычислений с плавающей точкой
6. **Обработка ошибок** - проверка исключения `ValueError` при делении на ноль
7. **Пограничные случаи** - специальные математические свойства (0 в степени 0 = 1, умножение на 0 = 0 и т.д.)

## Автоматизация тестирования

### Файл: `.github/workflows/test.yml`

Создан отдельный workflow для автоматического запуска тестов при каждом push в репозиторий.

#### Шаги CI/CD:
1. **Checkout code** - получение кода из репозитория
2. **Setup Python** - настройка Python 3.11
3. **Install dependencies** - установка зависимостей (если есть)
4. **Run tests** - автоматический запуск всех тестов с подробным выводом

#### Триггер:
- Запускается автоматически при каждом `push` в репозиторий

#### Проверка CI/CD:
После коммита файла `test.yml` в GitHub → Actions можно увидеть:
- Автоматический запуск тестов
- Результаты выполнения (✅ успешно или ❌ с ошибками)
- Детальный лог выполнения каждого шага
